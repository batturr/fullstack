# Tailwind CSS MCQ - Set 2 (Beginner Level)

**1. Which variant applies styles when the user hovers a pointer over an element?**

a) `pointer:`
b) `hover:`
c) `onhover:`
d) `h:`

**Answer: b) `hover:`**

---

**2. Which variant targets keyboard focus on focusable elements?**

a) `active:`
b) `focus:`
c) `tab:`
d) `key:`

**Answer: b) `focus:`**

---

**3. Which variant applies while a button is being pressed/activated?**

a) `pressed:`
b) `active:`
c) `click:`
d) `down:`

**Answer: b) `active:`**

---

**4. What does this class do?**

```html
<button class="bg-blue-600 hover:bg-blue-700">Save</button>
```

a) `Button is always darker blue`
b) `Background becomes darker blue on hover`
c) `Hover disables the button`
d) `Hover changes text color only`

**Answer: b) `Background becomes darker blue on hover`**

---

**5. Which class enables CSS Grid layout on an element?**

a) `grid`
b) `grids`
c) `layout-grid`
d) `display-grid`

**Answer: a) `grid`**

**6. Which utility creates a 3-column grid on large screens in this snippet?**

```html
<div class="grid grid-cols-1 md:grid-cols-3 gap-4">...</div>
```

a) `Always 1 column`
b) `1 column on small screens, 3 columns from `md` up`
c) `Always 3 columns`
d) `3 columns only on small screens`

**Answer: b) `1 column by default; 3 columns from the md breakpoint up`**
