# Bootstrap MCQ - Set 1 (Beginner Level)

**1. What is Bootstrap primarily described as?**

a) `A Java-only UI framework`
b) `A CSS and JavaScript toolkit for building responsive sites`
c) `A Python templating engine`
d) `A database migration tool`

**Answer: b) `A CSS and JavaScript toolkit for building responsive sites`**

---

**2. Which link tag correctly includes Bootstrap 5 CSS from a CDN?**

a) `<link rel="stylesheet" href="bootstrap.js">`
b) `<script src="bootstrap.min.css"></script>`
c) `<link href="https://cdn.jsdelivr.net/npm/bootstrap@5/dist/css/bootstrap.min.css" rel="stylesheet">`
d) `<link rel="icon" href="bootstrap.css">`

**Answer: c) `<link href="https://cdn.jsdelivr.net/npm/bootstrap@5/dist/css/bootstrap.min.css" rel="stylesheet">`**

---

**3. Where should Bootstrap’s JavaScript bundle typically be placed for correct behavior?**

a) `Only inside <head> before CSS`
b) `Before the opening <html> tag`
c) `Near the end of <body> (after content) or with defer in <head>`
d) `Inside a comment so it never runs`

**Answer: c) `Near the end of <body> (after content) or with defer in <head>`**
