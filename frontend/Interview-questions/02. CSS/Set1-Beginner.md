# CSS MCQ - Set 1 (Beginner Level)

**1. Which symbol is used to separate a CSS property from its value?**

a) `:`
b) `;`
c) `=`
d) `,`

**Answer: a) `:`**

---

**2. What is the correct syntax for an external stylesheet link in HTML?**

a) `<style href="styles.css">`
b) `<link rel="stylesheet" href="styles.css">`
c) `<css src="styles.css">`
d) `<script href="styles.css">`

**Answer: b) `<link rel="stylesheet" href="styles.css">`**

---

**3. Which selector targets all elements of type `p`?**

a) `.p`
b) `#p`
c) `p`
d) `*p`

**Answer: c) `p`**

---

**4. How do you select an element with `id="header"`?**

a) `.header`
b) `#header`
c) `header`
d) `id(header)`

**Answer: b) `#header`**

---

**5. How do you select elements with `class="btn"`?**

a) `#btn`
b) `.btn`
c) `btn`
d) `class.btn`

**Answer: b) `.btn`**

---

**6. Which selector matches every element in the document?**

a) `all`
b) `*`
c) `html *`
d) `:root *`

**Answer: b) `*`**

---

**7. How do you group selectors to share the same declarations?**

a) Separate with spaces: `h1 h2`
b) Separate with commas: `h1, h2`
c) Separate with `+`: `h1 + h2`
d) Separate with `>`: `h1 > h2`

**Answer: b) Separate with commas: `h1, h2`**

---

**8. What does this inline style set the text color to?**

```css
style="color: tomato;"
```

a) A shade of red/orange named `tomato`
b) Invalid; color names are not allowed
c) Black
d) Transparent

**Answer: a) A shade of red/orange named `tomato`**

---

**9. Which hex color represents full red?**

a) `#00ff00`
b) `#ff0000`
c) `#0000ff`
d) `#ffffff`

**Answer: b) `#ff0000`**

---

**10. What is the alpha channel in `rgba(0, 128, 255, 0.5)`?**

a) `0`
b) `128`
c) `255`
d) `0.5`

**Answer: d) `0.5`**

---

**11. Which property sets the background color of an element?**

a) `color`
b) `bgcolor`
c) `background-color`
d) `fill`

**Answer: c) `background-color`**

---

**12. What does `background-repeat: no-repeat` do?**

a) Tiles the image infinitely
b) Stretches the image to fill the box
c) Shows the image once without tiling
d) Hides the background image

**Answer: c) Shows the image once without tiling**

---

**13. Which value positions a background image at the top-right corner?**

a) `top left`
b) `center center`
c) `top right`
d) `100% 0%` only

**Answer: c) `top right`**

---

**14. What does `background-size: cover` typically do?**

a) Shows the image at its intrinsic size only
b) Scales the image to cover the entire area, may crop
c) Repeats the image to fill the area
d) Hides overflow of the image

**Answer: b) Scales the image to cover the entire area, may crop**

---

**15. Which property horizontally aligns inline content inside a block?**

a) `vertical-align`
b) `text-align`
c) `align-items`
d) `float`

**Answer: b) `text-align`**

---

**16. What does `text-decoration: underline` do?**

a) Adds spacing between letters
b) Draws a line under the text
c) Makes text uppercase
d) Indents the first line

**Answer: b) Draws a line under the text**

---

**17. Which property changes text to uppercase?**

a) `text-transform: capitalize`
b) `text-transform: uppercase`
c) `font-variant: uppercase`
d) `text-style: upper`

**Answer: b) `text-transform: uppercase`**

---

**18. What does `text-indent: 2em` affect?**

a) All lines are indented
b) Only the first line of a block is indented
c) Only the last line
d) Margin on the left of the box

**Answer: b) Only the first line of a block is indented**

---

**19. Which property sets space between letters?**

a) `word-spacing`
b) `letter-spacing`
c) `line-height`
d) `text-gap`

**Answer: b) `letter-spacing`**

---

**20. Which font stack is valid?**

a) `font-family: Arial, sans-serif;`
b) `font-family: Arial sans-serif;`
c) `font-family: "Arial sans-serif";`
d) `font: Arial, sans-serif;`

**Answer: a) `font-family: Arial, sans-serif;`**

---

**21. What is the effect of `font-weight: bold`?**

a) Italic text
b) Bolder glyph weight when available
c) Underlined text
d) Larger font size automatically

**Answer: b) Bolder glyph weight when available**

**22. Which property sets italic style?**

b) `font-style: italic`
c) `text-style: italic`

**Answer: b) `font-style: italic`**

**23. In the box model, which is outermost by default?**

a) `padding`
b) `border`
c) `margin`
d) `content`

**Answer: c) `margin`**

---

**24. What does `padding: 10px 20px` set?**

a) Top/bottom 20px, left/right 10px
b) Top/bottom 10px, left/right 20px
c) All sides 15px
d) Only top 10px and bottom 20px

**Answer: b) Top/bottom 10px, left/right 20px**

---

**25. Which shorthand sets width, style, and color of all borders?**

a) `outline`
b) `border`
c) `box-border`
d) `stroke`

**Answer: b) `border`**

---

**26. With default `box-sizing`, if `width: 200px` and `padding: 20px`, the total horizontal space used by content+padding inside the border is:**

a) `200px`

c) `240px`

**Answer: c) `240px`**

**30. How do you write a CSS comment?**

**31. Which unit is absolute and roughly 1/96th of an inch in CSS?**

a) `em`
b) `rem`

d) `%`

**Answer: c) `px`**

---

**32. Relative to what is `em` typically calculated for `font-size` on an element?**

a) Root element font size only
b) Viewport width
c) Parent element's computed font size
d) User agent default only

**Answer: c) Parent element's computed font size**

---

**33. Relative to what is `rem` calculated?**

a) Parent width
b) Root (`html`) element font size
c) Viewport height
d) Line height of parent

**Answer: b) Root (`html`) element font size**

---

**34. Percentage widths on a block-level element are generally relative to:**

a) The viewport only
b) The containing block's width
c) The element's own content width
d) Parent's height

**Answer: b) The containing block's width**

**35. What does `100vw` represent?**

a) 100% of the parent width
b) 100% of the viewport width

**Answer: b) 100% of the viewport width**

**36. What does `50vh` represent?**

a) Half the element's height

**Answer: b) Half the viewport height**

**37. Which rule turns all `strong` elements red?**

```css
strong {
  color: red;
}
```

a) Invalid; `strong` cannot be styled
b) Valid; `strong` text will be red
c) Only works inside `<style scoped>`

**47. Which is a valid declaration ending?**


**48. `background-image: url("pic.png")` expects:**

**49. `display: inline` elements typically:**

a) Start on a new line and fill width
b) Flow within line boxes; width/height often ignored for non-replaced elements

**50. Combining external and internal CSS, which wins if specificity is equal and internal comes later in the document?**

a) External always wins
b) Internal rule later in cascade wins
c) First rule wins
d) Inline styles never win

**Answer: b) Internal rule later in cascade wins**
