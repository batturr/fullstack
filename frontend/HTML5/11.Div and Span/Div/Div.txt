DIV
===
• The <div> tag defines a division or container or a section in an HTML document.
• Inside <div> tag you can place any content like normal text or any other html tags.
• When you want to divide your web page as no. of parts, each part is represented as <div> tag. 
• The <div> tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript.
• The <div> tag is easily styled by using the class or id attribute.
• Any sort of content can be put inside the <div> tag! 
• <div> is a paired tag.
• Always start on a new line
• Take full width available
• You can set width, height, margin, padding

• Example: <div>, <p>, <h1>, <section>, <ul>, <li>

• Example: <div> Hello </div>, <div>, <p>, <h1>, <section>, <ul>, <li>

• Note: By default, browsers always place a line break before and after the <div> element.

Syntax:
    <div>
        Your content here
    </div>

Example: 

<div> 
    Hello 
</div>
