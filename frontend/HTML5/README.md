# Advanced HTML5 Course Notes

A comprehensive 30-topic course covering everything from HTML fundamentals to modern HTML5 APIs and best practices.

---

## Course Index

| # | Topic | Description |
|---|-------|-------------|
| 01 | [Fundamentals](./01.Fundamentals/) | Web concepts, client-server architecture, browsers, HTTP |
| 02 | [HTML Basics](./02.HTML%20Basics/) | Introduction, versions, tags, syntax, DOCTYPE |
| 03 | [Basic Tags in HTML](./03.Basic%20Tags%20in%20HTML/) | Headings, paragraphs, line breaks, comments |
| 04 | [Text Formatting Tags](./04.Text%20Formatting%20Tags/) | Bold, italic, underline, strikeout, superscript, subscript, semantic vs presentational |
| 05 | [Images](./05.Images/) | img tag, responsive images, picture element, srcset, lazy loading, figure/figcaption |
| 06 | [Hyperlinks](./06.Hyperlinks/) | Anchor tag, link types, target, download, email/tel links, bookmarks |
| 07 | [List Tags](./07.List%20Tags/) | Ordered, unordered, description lists, nested lists |
| 08 | [Tables](./08.Tables/) | Table structure, colspan/rowspan, thead/tbody/tfoot, caption, scope, responsive tables |
| 09 | [Miscellaneous Tags](./09.Miscellaneous%20Tags/) | Blockquote, code, pre, abbr, address, mark, meter, time, wbr, dialog |
| 10 | [Forms](./10.Forms/) | All input types, form attributes, validation, fieldset, label, textarea, select |
| 11 | [Div and Span](./11.Div%20and%20Span/) | Block vs inline elements, use cases, semantic alternatives |
| 12 | [Advanced Tags](./12.Advanced%20Tags/) | Details/summary, datalist, progress, output, template, dialog |
| 13 | [Audio](./13.Audio/) | Audio tag, formats, attributes, autoplay policies, Web Audio API |
| 14 | [Video](./14.Video/) | Video tag, formats, attributes, tracks/captions, responsive video |
| 15 | [Semantic Tags](./15.Semantic%20Tags/) | Header, nav, main, section, article, aside, footer, figure, time |
| 16 | [Storage](./16.Storage/) | Local storage, session storage, IndexedDB comparison, cookies |
| 17 | [Geo Location](./17.Geo%20Location/) | Geolocation API, permissions, watchPosition, error handling |
| 18 | [Web Workers](./18.Web%20Workers/) | Dedicated workers, shared workers, service workers |
| 19 | [Drag and Drop](./19.Drag%20and%20Drop/) | Drag/drop API, events, dataTransfer, sortable lists |
| 20 | [Canvas](./20.Canvas/) | 2D context, shapes, paths, text, images, gradients, animations |
| 21 | [SVG](./21.SVG/) | Shapes, text, filters, animations, inline SVG, SVG vs Canvas |
| 22 | [Deprecated Tags & Attributes](./22.Deprecated%20Tags-Attributes/) | Removed elements, modern replacements |
| 23 | [Meta Tags & SEO](./23.Meta%20Tags%20%26%20SEO/) | Meta tags, Open Graph, Twitter Cards, structured data, robots |
| 24 | [Accessibility (a11y)](./24.Accessibility/) | ARIA roles, landmarks, focus management, screen readers |
| 25 | [Iframes & Embedding](./25.Iframes%20%26%20Embedding/) | iframe, embed, object, sandbox, postMessage |
| 26 | [Web Components](./26.Web%20Components/) | Custom elements, shadow DOM, templates, slots |
| 27 | [HTML APIs](./27.HTML%20APIs/) | History, Clipboard, Fullscreen, Intersection Observer, Notifications |
| 28 | [Performance & Optimization](./28.Performance%20%26%20Optimization/) | Lazy loading, preload, prefetch, async/defer, resource hints |
| 29 | [Responsive HTML](./29.Responsive%20HTML/) | Viewport, picture, srcset, media in HTML |
| 30 | [Best Practices & Modern HTML](./30.Best%20Practices/) | Document outline, content models, security, CSP |
| — | [Differences](./Differences/) | HTML vs HTML5, semantic vs non-semantic, div vs span |
