# ðŸ”µ **1. What is `<fieldset>`?**

`<fieldset>` is a **form tag** used to **group related form elements** inside a form.

It places a **box (border)** around the grouped inputs to show they belong together.

---

# ðŸ”µ **2. Why do we use `<fieldset>`? (Purpose)**

* Groups related inputs
* Makes forms look organized
* Improves readability
* Adds a visible border around the section
* Helps screen readers â†’ improves accessibility
* Used for sections like **Personal Info**, **Login Info**, **Address**, etc.

---

# ðŸ”µ **3. Syntax**

```html
<fieldset>
   ... form controls ...
</fieldset>
```

---

# ðŸ”µ **4. `<legend>` (Title for Fieldset)**

`<legend>` gives a **heading** to the fieldset box.

```html
<fieldset>
    <legend>Personal Details</legend>
</fieldset>
```

---

# ðŸ”µ **5. Example with Inputs**

```html
<form>
  <fieldset>
    <legend>User Info</legend>

    Name: <input type="text"><br><br>
    Email: <input type="email"><br><br>
    Age: <input type="number">
  </fieldset>
</form>
```

---

# ðŸ”µ **6. Attributes of `<fieldset>`**

| Attribute  | Use                                     |
| ---------- | --------------------------------------- |
| `disabled` | Disables all inputs inside the fieldset |
| `form`     | Associates fieldset with a form         |
| `name`     | Gives a name to the fieldset            |

### Example:

```html
<fieldset disabled>
    <legend>Disabled Section</legend>
    <input type="text" value="You cannot edit this">
</fieldset>
```
