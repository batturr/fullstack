Difference between Div and Span
                Div Tag														Span Tag
1. The <div> tag is a block level element.				1. The <span> tag is an inline element.
2. It is used to group large sections of content.		2. It is used to group small parts of text.
3. It starts on a new line.								3. It stays on the same line.
4. It accepts align attribute.							4. It does not accept align attribute.
5. It is used while creating CSS based layouts in html.	5. It is uses to stylize text.
