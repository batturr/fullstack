BOOTSTRAP 5.3+ - TEXT COLORS
=============================

Overview
--------
• Used to set the color of text content
• Provides predefined semantic color classes for different purposes
• Based on the requirement, use any available color class
• Can customize using CSS variables or custom CSS

Text Color Classes (Bootstrap 5.3+)
===================================

Primary Colors:
• text-primary       : Blue (#0d6efd) - Main action/attention
• text-secondary     : Gray (#6c757d) - Secondary information
• text-success       : Green (#198754) - Positive/Success
• text-danger        : Red (#dc3545) - Error/Negative
• text-warning       : Orange (#ffc107) - Warnings/Alerts
• text-info          : Cyan (#0dcaf0) - Information/Help

Special Text Colors:
• text-dark          : Dark gray (#212529) - Dark text
• text-light         : Light gray (#f8f9fa) - Light text (use on dark backgrounds)
• text-body          : Default (#212529) - Body text color
• text-muted         : Gray (#6c757d) - Muted/disabled text
• text-white         : White (#ffffff) - White text
• text-white-50      : White with 50% opacity - Semi-transparent white
• text-black-50      : Black with 50% opacity - Semi-transparent black

Syntax Examples
===============

Basic Text Colors:
  <p class="text-primary">Primary text color</p>
  <p class="text-success">Success message</p>
  <p class="text-danger">Error message</p>
  <p class="text-warning">Warning message</p>
  <p class="text-info">Information text</p>
  <p class="text-secondary">Secondary text</p>
  <p class="text-dark">Dark text</p>
  <p class="text-light bg-dark p-2">Light text on dark background</p>
  <p class="text-muted">Muted/disabled text</p>

Opacity Variants (NEW in Bootstrap 5.3+):
=========================================
Bootstrap 5.3+ introduces opacity control without needing custom CSS:

• text-opacity-100   : Full opacity (100%)
• text-opacity-75    : 75% opacity
• text-opacity-50    : 50% opacity
• text-opacity-25    : 25% opacity

Usage:
  <p class="text-primary text-opacity-100">100% opacity</p>
  <p class="text-primary text-opacity-75">75% opacity</p>
  <p class="text-primary text-opacity-50">50% opacity</p>
  <p class="text-primary text-opacity-25">25% opacity</p>

2. text-success

3. text-danger
   • Error messages
   • Validation failures

4. text-warning
   • Warning messages

5. text-info

Bootstrap 5.3+ CSS Variables
=============================
All colors are defined as CSS custom properties:

:root {
  --bs-primary: #0d6efd;
  --bs-secondary: #6c757d;
  --bs-success: #198754;
  --bs-danger: #dc3545;
  --bs-warning: #ffc107;
  --bs-info: #0dcaf0;
  --bs-light: #f8f9fa;
  --bs-dark: #212529;
  --bs-white: #ffffff;
}

Customize colors by overriding CSS variables:
  <style>
    :root {
      --bs-primary: #ff6b6b;     /* Change primary to custom color */
      --bs-success: #51cf66;     /* Change success to custom color */
    }
  </style>

Dark Mode Support (Bootstrap 5.3+)
==================================
Bootstrap 5.3+ includes built-in dark mode support:

Enable dark mode:
  <html data-bs-theme="dark" lang="en">

In dark mode, colors automatically adjust for better contrast:
• Backgrounds become darker
• Text becomes lighter
• All colors remain accessible

Complete Example
================

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Text Colors Example</title>

    <p class="text-info"><i class="bi bi-info-circle"></i> New features available</p>
    
    <!-- Opacity variants -->

    <p class="text-primary text-opacity-25">25% opacity primary text</p>
  </div>
  
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>

Accessibility & Best Practices
===============================

✓ DO:
  • Use semantic colors (primary for main actions, danger for errors)
  • Maintain sufficient contrast ratio (4.5:1 for normal text)
  • Combine colors with icons or text labels
  • Test with color blindness simulators
  • Use opacity variants for subtle emphasis

✗ DON'T:
  • Rely solely on color to convey information
  • Use low contrast combinations
  • Mix too many different colors
  • Ignore accessibility standards (WCAG)
  • Override colors without purpose

Key Improvements from Bootstrap 4 to 5.3+
==========================================

1. Opacity utilities - Easy opacity control without custom CSS
2. CSS variables - All colors customizable via CSS custom properties
3. Dark mode - Built-in dark mode support
4. Better contrast - Improved WCAG AA/AAA compliance
5. New color utilities - text-body-secondary, etc.

Migration Notes
===============

Bootstrap 4 → Bootstrap 5.3+
=============================
• .text-muted → .text-muted (unchanged, still works)
• No changes to basic text-primary, text-success, etc.
• NEW: text-opacity-25/50/75 classes available
• NEW: data-bs-theme attribute for dark mode
