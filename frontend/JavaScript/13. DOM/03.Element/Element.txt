The Element Object
===================
In the HTML DOM, the Element object represents an HTML element, like P, DIV, A, TABLE, or any other HTML element.

Properties and Methods
=================================
The following properties and methods can be used on all HTML elements.

1.accessKey	            => Sets or returns the accesskey attribute of an element
2.addEventListener()	=> Attaches an event handler to an element
3.appendChild()	        => Adds (appends) a new child node to an element
4.attributes	        => Returns a NamedNodeMap of an element's attributes
5.blur()	            => Removes focus from an element
6.childElementCount	    => Returns an elements's number of child elements
7.childNodes	        => Returns a NodeList of an element's child nodes
8.children	            => Returns an HTMLCollection of an element's child elements
9.classList	            => Returns the class name(s) of an element
10.className	        => Sets or returns the value of the class attribute of an element
11.click()	            => Simulates a mouse-click on an element
12.clientHeight	        => Returns the height of an element, including padding
13.clientLeft	        => Returns the width of the left border of an element
14.clientTop	        => Returns the width of the top border of an element
15.clientWidth	        => Returns the width of an element, including padding
16.cloneNode()	        => Clones an element
17.closest()	        => Searches the DOM tree for the closest element that matches a CSS selector
18.compareDocumentPosition() => Compares the document position of two elements
19.contains()	        => Returns true if a node is a descendant of a node
20.contentEditable	    => Sets or returns whether the content of an element is editable or not
21.dir	                => Sets or returns the value of the dir attribute of an element
22.firstChild	        => Returns the first child node of an element
23.firstElementChild	=> Returns the first child element of an element
24.focus()	            => Gives focus to an element
25.getAttribute()	    => Returns the value of an element's attribute
26.getAttributeNode()	=> Returns an attribute node
27.getBoundingClientRect()	=> Returns the size of an element and its position relative to the viewport
28.getElementsByClassName()	=> Returns a collection of child elements with a given class name
29.getElementsByTagName()	=> Returns a collection of child elements with a given tag name
30.hasAttribute()       => Returns true if an element has a given attribute
31.hasAttributes()      => Returns true if an element has any attributes
32.hasChildNodes()      => Returns true if an element has any child nodes
33.id	                => Sets or returns the value of the id attribute of an element
34.innerHTML	        => Sets or returns the content of an element
35.innerText	        => Sets or returns the text content of a node and its descendants
36.insertAdjacentElement() => Inserts a new HTML element at a position relative to an element
37.insertAdjacentHTML()	=> Inserts an HTML formatted text at a position relative to an element
38.insertAdjacentText()	=> Inserts text into a position relative to an element
39.insertBefore()	    => Inserts a new child node before an existing child node
40.isContentEditable	=> Returns true if an element's content is editable
41.isDefaultNamespace()	=> Returns true if a given namespaceURI is the default
42.isEqualNode()	    => Checks if two elements are equal
43.isSameNode()	        => Checks if two elements are the same node
44.lang	                => Sets or returns the value of the lang attribute of an element
45.lastChild	        => Returns the last child node of an element
46.lastElementChild	    => Returns the last child element of an element
47.matches()	        => Returns true if an element is matched by a given CSS selector
48.namespaceURI	        => Returns the namespace URI of an element
49.nextSibling	        => Returns the next node at the same node tree level
50.nextElementSibling	=> Returns the next element at the same node tree level
51.nodeName	            => Returns the name of a node
52.nodeType	            => Returns the node type of a node
53.nodeValue	        => Sets or returns the value of a node
54.normalize()	        => Joins adjacent text nodes and removes empty text nodes in an element
55.offsetHeight	        => Returns the height of an element, including padding, border and scrollbar
56.offsetWidth	        => Returns the width of an element, including padding, border and scrollbar
57.offsetLeft	        => Returns the horizontal offset position of an element
58.offsetParent	        => Returns the offset container of an element
59.offsetTop	        => Returns the vertical offset position of an element
60.outerHTML	        => Sets or returns the content of an element (including the start tag and the end tag)
61.outerText	        => Sets or returns the outer text content of a node and its descendants
62.ownerDocument	    => Returns the root element (document object) for an element
63.parentNode	        => Returns the parent node of an element
64.parentElement	    => Returns the parent element node of an element
65.previousSibling	    => Returns the previous node at the same node tree level
66.previousElementSibling	=> Returns the previous element at the same node tree level
67.querySelector()	    => Returns the first child element that matches a CSS selector(s)
68.querySelectorAll()	=> Returns all child elements that matches a CSS selector(s)
69.remove()	            => Removes an element from the DOM
70.removeAttribute()	=> Removes an attribute from an element
71.removeAttributeNode()=> 	Removes an attribute node, and returns the removed node
72.removeChild()        => 	Removes a child node from an element
73.removeEventListener()=> 	Removes an event handler that has been attached with the addEventListener() method
74.replaceChild()	    => Replaces a child node in an element
75.scrollHeight	        => Returns the entire height of an element, including padding
76.scrollIntoView()     => Scrolls the an element into the visible area of the browser window
77.scrollLeft	        => Sets or returns the number of pixels an element's content is scrolled horizontally
78.scrollTop	        => Sets or returns the number of pixels an element's content is scrolled vertically
79.scrollWidth	        => Returns the entire width of an element, including padding
80.setAttribute()	    => Sets or changes an attribute's value
81.setAttributeNode()	=> Sets or changes an attribute node
82.style	            => Sets or returns the value of the style attribute of an element
83.tabIndex	            => Sets or returns the value of the tabindex attribute of an element
84.tagName	            => Returns the tag name of an element
85.textContent      	=> Sets or returns the textual content of a node and its descendants
86.title	            => Sets or returns the value of the title attribute of an element
87.toString()	        => Converts an element to a string
