## Step 1: Download Visual Studio Code

1. Open your web browser (Chrome, Firefox, Edge, etc.)
2. Navigate to the official VS Code website: [https://code.visualstudio.com](https://code.visualstudio.com)
3. Click on the **"Download for Windows"** button
   - The website automatically detects your operating system
   - This will download the VS Code installer (`.exe` file)
4. Wait for the download to complete
   - File name: `VSCodeUserSetup-{version}.exe`
   - File size: Approximately 80-100 MB

---

## Step 2: Run the Installer

1. Locate the downloaded installer file in your **Downloads** folder
2. Double-click on **`VSCodeUserSetup-{version}.exe`** to launch the installer
3. If prompted by User Account Control (UAC), click **"Yes"** to allow the installer to make changes

---

## Step 3: Accept the License Agreement

1. Read the **License Agreement**
2. Select **"I accept the agreement"**
3. Click **"Next"** to continue

---

## Step 4: Choose Installation Location

1. The default installation path is: `C:\Users\{YourUsername}\AppData\Local\Programs\Microsoft VS Code`
2. You can keep the default location or click **"Browse"** to choose a different location
3. Click **"Next"** to continue

---

## Step 5: Select Start Menu Folder

1. The default Start Menu folder is **"Visual Studio Code"**
2. Keep the default or change it as needed
3. Click **"Next"** to continue

---

---

## Step 7: Install

1. Review your installation settings
2. Click **"Install"** to begin the installation
3. Wait for the installation to complete (usually takes 1-2 minutes)
4. Progress bar will show the installation status

---

## Step 8: Complete Installation

1. Once installation is complete, you'll see the **"Completing the Visual Studio Code Setup Wizard"** screen
2. Check the box **"Launch Visual Studio Code"** if you want to open VS Code immediately
3. Click **"Finish"** to complete the setup

---

## Step 9: Launch Visual Studio Code

### Option 1: From Start Menu
1. Click on **Start** button
2. Type **"Visual Studio Code"** in the search bar
3. Click on the **Visual Studio Code** app

### Option 2: From Desktop
1. Double-click the **Visual Studio Code** icon on your desktop (if you selected this option during installation)

### Option 3: From Command Line/Terminal
1. Open Command Prompt or PowerShell
2. Type: `code`
3. Press **Enter**

---

## Step 10: Initial Setup (Optional)

When you first launch VS Code:

1. **Welcome Screen** will appear with options:
   - Open a folder
   - Clone a Git repository
   - Install extensions
   - Customize settings

2. **Choose a Theme** (optional):
   - Click on **Settings** icon (bottom-left corner)
   - Select **Color Theme**
   - Choose from Light, Dark, or High Contrast themes

3. **Install Extensions** (optional but recommended for JavaScript):
   - Click on **Extensions** icon in the left sidebar (or press `Ctrl+Shift+X`)
   - Search and install:
     - **ESLint** - JavaScript linting
     - **Prettier** - Code formatter
     - **JavaScript (ES6) code snippets** - Code snippets
     - **Live Server** - Launch development local server

## System Requirements

### Minimum Requirements:
- **OS**: Windows 10 or Windows 11 (64-bit)
- **Processor**: 1.6 GHz or faster
- **RAM**: 1 GB (2 GB recommended)
- **Disk Space**: 200 MB

### Recommended:
- **OS**: Windows 10/11 (64-bit)
- **Processor**: Multi-core processor
- **RAM**: 4 GB or more
- **Disk Space**: 500 MB or more

## Troubleshooting

### Issue: VS Code won't launch
**Solution**: 
- Restart your computer
- Reinstall VS Code
- Check if antivirus is blocking the application

### Issue: "code" command not recognized in terminal
**Solution**:
- Reinstall VS Code and ensure **"Add to PATH"** option is selected
- Restart your terminal/command prompt
- Reboot your computer

### Issue: Extensions won't install
**Solution**:
- Check your internet connection
- Disable firewall/proxy temporarily
- Clear VS Code cache: `%APPDATA%\Code` â†’ delete the folder
