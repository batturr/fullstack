## JavaScript Hello World Example

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>JavaScript Example</title>
</head>
    <body>
        <h1>JavaScript Hello World Example</h1>
        <script>
            console.log("Welcome to JavaScript world.");
        </script>
    </body>
</html>
```

## Save the File

- Go to **File** menu ‚Üí **Save** (or) Press `Ctrl+S`.
- Go to `c:\` and click on **New folder**. Enter the new folder name as **"ui"**.
- Select `c:\ui` folder and enter the filename as **"first.html"**.
- Click on **Save**.
- Now the file (`c:\ui\first.html`) is ready.

## Code Explanation

Let's understand what each part of the code does:

```html
<!DOCTYPE html>
```
- Declares that this is an HTML5 document

```html
<html lang="en">
```
- Root element, `lang="en"` specifies English language

```html
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>JavaScript Example</title>
</head>
```
- **`<head>`**: Contains metadata about the document
- **`charset="UTF-8"`**: Sets character encoding for special characters
- **`viewport`**: Makes the page responsive on mobile devices
- **`<title>`**: Sets the browser tab title

```html
<body>
    <h1>JavaScript Hello World Example</h1>
    <script>
        console.log("Welcome to JavaScript world.");
    </script>
</body>
```
- **`<h1>`**: Displays heading on the webpage
- **`<script>`**: Contains JavaScript code
- **`console.log()`**: Prints message to browser console (for developers)

## Troubleshooting

### Problem: Console message not showing
**Solution:**
- Make sure you pressed **F12** to open Developer Tools
- Click on the **"Console"** tab
- Refresh the page (`F5` or `Ctrl+R`)

### Problem: Blank page in browser
**Solution:**
- Check if the file was saved properly
- Look for any typos in the HTML tags
- Make sure all opening tags have closing tags

### Problem: "Cannot find file" error
**Solution:**
- Verify the file path is correct
- Make sure the file has `.html` extension, not `.txt`
- Check that the file is saved before opening

### Problem: Script not running
**Solution:**
- Ensure `<script>` tag is properly closed with `</script>`
- Check browser console for error messages (they'll be in red)

## Try These Variations

### Example 1: Show Alert Box
```html
<script>
    console.log("Welcome to JavaScript world.");
    alert("Hello, JavaScript!");
</script>
```

### Example 2: Display in Webpage
```html
<body>
    <h1>JavaScript Hello World Example</h1>
    <p id="demo"></p>
    
    <script>
        console.log("Welcome to JavaScript world.");
        document.getElementById("demo").innerHTML = "JavaScript is running!";
    </script>
</body>
```

### Example 3: Button Click
```html
<body>
    <h1>JavaScript Hello World Example</h1>
    <button onclick="myFunction()">Click Me!</button>
    <p id="result"></p>
    
    <script>
        function myFunction() {
            document.getElementById("result").innerHTML = "Button was clicked!";
            console.log("Button clicked!");
        }
    </script>
</body>
```

## Best Practices

‚úÖ **Do:**
- Always save your file before viewing in browser
- Use meaningful file names (`calculator.html`, `form-validation.html`)
- Keep your code properly indented for readability
- Test in multiple browsers (Chrome, Firefox, Edge)
- Check the console for errors regularly

‚ùå **Don't:**
- Use spaces in file names (use `my-file.html` not `my file.html`)
- Forget to close HTML tags
- Put `<script>` before elements you want to manipulate
- Ignore console error messages
