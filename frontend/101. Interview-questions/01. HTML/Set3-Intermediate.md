# HTML MCQ - Set 3 (Intermediate Level)

**1. Which element wraps the primary navigation links of a site?**

a) `<menu>`
b) `<nav>`
c) `<navigation>`
d) `<header>` only

**Answer: b) `<nav>`**

---

**2. Which element represents introductory or grouping content at the top of a section or page?**

a) `<top>`
b) `<header>`
c) `<head>`
d) `<banner>`

**Answer: b) `<header>`**

---

**3. Which element typically contains footer information for its nearest sectioning ancestor or the page?**

a) `<bottom>`
b) `<footer>`
c) `<aside>`
d) `<end>`

**Answer: b) `<footer>`**

---

**4. Which element should wrap the dominant content unique to a document (use once per page typically)?**

a) `<body>`
b) `<main>`
c) `<article>`
d) `<section>` only

**Answer: b) `<main>`**
