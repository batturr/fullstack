# HTML MCQ - Set 1 (Beginner Level)

**1. What does `<!DOCTYPE html>` declare?**

a) `The page uses HTML4`
b) `The document is an HTML5 document`
c) `The page requires JavaScript`
d) `The charset is UTF-8`

**Answer: b) `The document is an HTML5 document`**

---

**2. Which element is the root of an HTML document?**

a) `<head>`
b) `<body>`
c) `<html>`
d) `<document>`

**Answer: c) `<html>`**

---

**3. Where should metadata such as `<title>` and `<meta>` typically be placed?**

a) `Inside <body>`
b) `Inside <head>`
c) `After </html>`
d) `Inside <footer>`

**Answer: b) `Inside <head>`**

---

**4. What is the correct order for a minimal valid HTML5 document structure?**

a) `<html>`, then `<body>`, then `<head>`
b) `<head>`, then `<html>`, then `<body>`
c) `<html>` containing `<head>` then `<body>`
d) `<body>` containing `<head>`

**Answer: c) `<html>` containing `<head>` then `<body>`**

---

**5. Which heading tag represents the highest (most important) level by default?**

a) `<h6>`
b) `<h1>`
c) `<heading>`
d) `<title>`

**Answer: b) `<h1>`**

---

**6. What will this render as?**

```html
<h3>Section</h3>
<h2>Chapter</h2>
```

a) `Both same size`
b) `h3 larger than h2`
c) `h2 larger than h3 by default`
d) `Invalid HTML`

**Answer: c) `h2 larger than h3 by default`**

**13. What does the `href` attribute on `<a>` specify?**

a) `Link target window name only`
b) `The URL or destination of the hyperlink`

**14. What does `target="_blank"` typically do?**

a) `Submits the form in a new tab`

**Answer: b) `Opens the linked document in a new browsing context`**

**16. What is wrong with this image tag for valid HTML5?**

```html

```
