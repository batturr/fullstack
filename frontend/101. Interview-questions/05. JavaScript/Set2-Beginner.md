# JavaScript MCQ - Set 2 (Beginner Level)

**1. What does `"hello".toUpperCase()` return?**

a) `"Hello"`
b) `"HELLO"`
c) `"hello"`
d) `undefined`

**Answer: b) `"HELLO"`**

**43. What does this log?

```javascript
console.log(x);
var x = 1;
```**

b) `undefined`
c) Throws ReferenceError
d) `null`

**Answer: b) `undefined`** *(var is hoisted; assignment is not)*

**44. What happens here?

```javascript
console.log(y);
let y = 1;
```**

a) Logs `undefined`
b) Logs `1`
c) Throws a ReferenceError (temporal dead zone)
d) Logs `null`

**Answer: c) Throws a ReferenceError (temporal dead zone)**

**45. A **function declaration** can be called before it appears in the source file because:**

a) Declarations are hoisted with their full body

c) Only arrow functions are hoisted
d) The engine always runs files bottom-to-top

**Answer: a) Declarations are hoisted with their full body**
