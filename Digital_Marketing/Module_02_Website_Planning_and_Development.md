### 2.4.2 High-Converting Landing Page Anatomy

## 2.5 Website Performance & Technical Optimization

### 2.5.1 Page Speed Optimization
- Why speed matters: impact on SEO, conversions, and user experience
- Core Web Vitals: LCP (Largest Contentful Paint), FID/INP (Interaction to Next Paint), CLS (Cumulative Layout Shift)
- Image optimization: formats (WebP, AVIF), compression, lazy loading
- Code minification: HTML, CSS, JavaScript
- Browser caching strategies
- Critical CSS and above-the-fold rendering
- Reducing HTTP requests
- Font optimization and font-display strategies
- Tools: Google PageSpeed Insights, GTmetrix, WebPageTest, Lighthouse

### 2.5.2 Website Security for Marketers
- SSL/TLS certificates (Let's Encrypt, paid certificates)
- Common threats: malware, SQL injection, XSS, DDoS
- Security plugins and firewalls (Wordfence, Sucuri, Cloudflare WAF)
- Regular backups and disaster recovery
- Keeping CMS and plugins updated
- Two-factor authentication for admin access
- Security headers: CSP, HSTS, X-Frame-Options

### 2.5.3 Website Maintenance & Management
- Regular content updates and freshness
- Broken link monitoring (Screaming Frog, Ahrefs)
- Uptime monitoring (UptimeRobot, Pingdom)
- Database optimization
- Plugin/theme update management
- Staging environments for testing changes
- Version control basics for marketers

### 2.5.4 Schema Markup & Structured Data
- What is schema markup and why it matters for SEO
- Types: Organization, Product, FAQ, HowTo, Review, Event, Article, BreadcrumbList
- JSON-LD implementation (preferred by Google)
- Tools: Google Structured Data Markup Helper, Schema.org
- Testing: Rich Results Test, Schema Validator
- Impact on rich snippets and SERP features
